## Sections

1. **Hero Section** - Introduction with profile image and tagline
2. **About Section** - Brief bio and current position

4. **Skills Section** - Technology stack and expertise

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Google Fonts (Plus Jakarta Sans, Poppins)

## How to Use

1. Open `index.html` in a web browser
2. Customize the content with your own information
3. Replace the Figma asset URLs with your own images
4. Update colors in CSS custom properties if needed

## Customization

To change the color scheme, edit the CSS variables in `styles.css`:

```css
:root {
    --color-primary: #7127ba;
    --color-secondary: #9857d3;
    --color-dark: #1a0b2e;
    /* ... */
}
```
